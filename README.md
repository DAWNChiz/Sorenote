# StudyFlow 💜
### Human Rights MDC 3F — Ranchi University

A beautiful, motivating study note app for Human Rights (MDC 3F) students at Ranchi University under FYUGP.

## Features
- 📅 7-day structured study plan across all 5 units
- 📖 Pre-written, detailed notes for every topic
- ⏳ 30-second read timer before copying (so you actually read!)
- ✅ Progress tracking with completion badges
- 🔥 Motivational quotes and streak tracker
- 💜 Beautiful dark UI — works on mobile and desktop

## Topics Covered
- **Unit II:** UDHR, RTI, ICCPR, ICESCR, CEDAW, CRC
- **Unit III:** Fundamental Rights, DPSP, Fundamental Duties, Role of Judiciary
- **Unit IV:** NHRC, JSHRC, NGOs, Media, NCM, NCW
- **Unit V:** Challenges, Terrorism, Health/Environment, Discrimination & Violence

## Running Locally

```bash
npm run dev
```
